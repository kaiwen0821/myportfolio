JioMe is a campus-focused interest-group finder web application that helps students quickly create, discover, and join short, informal activity groups.

Problem

Students often rely on scattered chat messages to organize activities, which can lead to confusion, missed opportunities, and difficulty tracking availability.

Solution

JioMe allows users to create lobbies for activities, browse available lobbies with filters, join sessions with real-time slot tracking, and enforce ownership rules so only lobby creators can edit or cancel sessions.

Design Focus

Built with React, Node.js, and MySQL, the app focuses on real-time updates, a clear and structured interface, responsive design, and an intuitive user experience for campus students.

Key Features

Create/join lobbies, filter activities, real-time slot tracking, edit/delete only for owners, structured and centralized coordination

Tech Stack:

  • Frontend: React, JavaScript, HTML, CSS
  • Backend: Node.js, Express
  • Database: MySQL
  • Authentication: Token-based authentication (login returns token + user_id)
  • API: RESTful API (CRUD operations for users and lobbies)
  • Deployment: Vercel (Frontend & Backend)
  • Version Control: Git & GitHub (branching, PRs, merges)
