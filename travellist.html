My Travel List ✈️

My Travel List is a React-based web application that helps users plan and track their packing items. Users can add, mark as packed, delete, and restore items, with a visually engaging interface, all managed in real-time using React state without a backend database.

Tech Stack:

  • Frontend: React (with JSX), CSS
  • Data Storage: In-memory (arrays/objects)
