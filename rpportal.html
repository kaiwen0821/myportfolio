RP Course Enrolment Portal

RP Course Enrolment Portal is a React-based web application designed for Republic Polytechnic students to explore diplomas, view course details, and register interest efficiently.

Problem

Students need a clear and structured platform to navigate available courses and diplomas without confusion or scattered information.

Solution

The portal provides search, favourites, nested routes for smooth navigation, and registration forms for students to express interest in specific courses.

Design Focus

Built with React and React Router, the interface focuses on clarity, responsiveness, and smooth navigation while using React Hooks for state management and controlled forms.

Key Features

Diploma/course exploration, search and filter, favourites system, registration forms, responsive and clean UI

Tech Stack:

  • Frontend: React, JavaScript (JSX)
  • Routing: React Router (Nested Routes)
  • Styling: CSS / Custom Responsive Styling
  • State Management: React Hooks (useState, useEffect)
  • Forms: Controlled Components & Form Handling
