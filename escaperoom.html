Locked in the Library is a VR escape room adventure built in Unity Hub, set inside a mysterious library where players solve puzzles to escape.

Problem

Traditional escape room experiences are limited to physical spaces and cannot always offer interactive, immersive challenges in a safe or accessible environment.

Solution

This VR escape room lets players explore a digital library, uncover hidden clues, and solve four puzzles to unlock the exit. Interactions include finding keys, revealing numbers under objects, and entering the correct code on a keypad to escape.

Design Focus

Built with Unity Hub, the experience emphasizes immersion, intuitive puzzle mechanics, and a visually engaging environment, ensuring players feel challenged while exploring the virtual library.

Key Features

VR interaction, four unique puzzles, immersive library environment, visual clues, interactive objects, intuitive escape mechanics

Tech Stack:

  • Unity Hub
