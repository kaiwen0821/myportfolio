Locked in the library is a VR escape room adventure set inside a quiet, mysterious library. Players must solve four puzzles to break free. A key found on a table unlocks a drawer that vanishes to reveal the first number on a wall. A hint directs players to move a stack of books on a chair to uncover another hidden number. Lifting a cowboy hat on a table exposes a number formed using pens. After collecting all clues, players enter the full code into a keypad to escape the library. The entire experience was built in Unity Hub.
