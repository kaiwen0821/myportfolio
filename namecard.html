AR Name Card is an augmented reality project that transforms a traditional business card into a dynamic, interactive experience using Unity Hub.

Problem

Traditional business cards are static and cannot engage users beyond displaying contact information and branding.

Solution

The AR Name Card allows users to scan the card and view a 3D stylized butterfly logo emerge from the surface, creating a memorable and interactive networking experience.

Design Focus

Built with Unity Hub, the card combines vibrant design, clear branding, and seamless AR functionality to enhance user engagement and showcase technical proficiency in AR development.

Key Features

AR 3D object display, interactive branding, visually striking design, immersive networking experience, technical AR implementation

Tech Stack:

  • Unity Hub
