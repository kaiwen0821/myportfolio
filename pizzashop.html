KaiWen's Pizza Co. is a React-based web application that lets users explore pizza menus, mark favourites, and place orders during operating hours.

Problem

Users have difficulty checking pizza options and ordering efficiently without a clear, interactive interface.

Solution

The app provides dynamic favourite toggling, real-time open/close shop status, pizza details, and interactive components to enhance the user experience.

Design Focus

Built with React, the interface emphasizes a visually appealing, responsive layout with interactive components and real-time state updates for smooth user interactions.

Key Features

View pizza menu, mark favourites, check shop status, interactive and responsive interface

Tech Stack:

  • Frontend: React (with JSX), CSS
  • Data Storage: In-memory (arrays/objects)
